Delivery Driver

Description

Delivery Driver is set up to be a delivery game. The player would run around the map, collecting objects to delivery to specified locations. Watch out for buildings and obstacles, They will slow you down!


Delivery Driver is the first project I made in Unity. I do not think it deserves being called a video game - there is no game to play - but the project taught me the fundamentals of creating a Unity video game. Some of the concepts I learned are as follows:

  • How to add components to game objects
  • How to create and use scripts
  • How to gather user input and use that information to interact with game objects
  • Set up triggers and collisions
  • How to use rigidbodies

Link to Github: https://github.com/JeremyRoalef/Delivery-Driver


Snow Boarder

Description

Snow Boarder is a balancing game where you have to stay right side up if you want to live. Thats all...


Snow Boarder was the second project I worked on for Fall 2023. The game was rather simplistic in design, and the game is only one level long, but I learned new concepts and solidified old concepts

Concepts learned in Snow Boarder:

  • Creating parent and child objects
  • Working with edge colliders
  • Loading/reloading scenes
  • Using Unity's Particle System

Link to Github: https://github.com/JeremyRoalef/SnowBoarder


Quiz Master

Description

Quiz Master is a quiz game. Test your knowledge on basic programming concepts. You will be graded.


Quiz Master was an introduction to the user interface (UI). Before, my games had no UI and suffered greatly if you ask me. The addition of UI helps the feel of the game greatly.

Concepts learned in Quiz Master:

  • Creating canvases
  • Using buttons
  • Working with time
  • Creating scriptable objects

Link to Github: https://github.com/JeremyRoalef/QuizMaster


Tile Vania

Description

Tile Vania is a 2d platformer with the goal of reching the level's exit. Along your journey in Tile Vania, you will encounter enemies, abilities, obstacles, and death. Collect coins and try to reach a high score!


Tile Vania introduced a multitude of new and important concepts. The original game is very simple, so I took the liberty of using this game as a template for one of my final projects. I developed levels, abilities, effects, scene transitions, animations, sprites (arrows and elevators), particle systems, and obstacles for the game.

Concepts learned in Tile Vania:

  • Creating animations
  • Creating tilemaps
  • Creating enemy logic
  • Learned intricacies of previous concepts
  • Creating player abilities
  • How to work with a lot of logic
  • Playing with Unity's particle system further

Link to Github: https://github.com/JeremyRoalef/TileVania
